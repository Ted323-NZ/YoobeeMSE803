# Recording Guide for the 3-Minute Video

Use this guide together with `presentation_script.md`.

## Option 1 - QuickTime + PowerPoint or Keynote

1. Open `output/beijing_air_quality_insights_week2_activity2.pptx`.
2. Open `presentation_script.md` on a second screen or device.
3. Start a screen recording in QuickTime Player.
4. Present the slides and read the script naturally.
5. Stop the recording before 3 minutes.
6. Save the exported video in this folder, for example:
   - `output/week2_activity2_video.mov`

## Option 2 - PowerPoint Recording

1. Open the `.pptx` file in Microsoft PowerPoint.
2. Use the built-in slideshow recording feature.
3. Read the content from `presentation_script.md`.
4. Export the recording as a video file.

## Suggested Timing

- Slide 1: 20 seconds
- Slide 2: 25 seconds
- Slide 3: 35 seconds
- Slide 4: 35 seconds
- Slide 5: 35 seconds
- Slide 6: 25 seconds

Total: about 2 minutes 55 seconds
